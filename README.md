# 📊 Retail Sales Analysis

## 🚀 Overview
This project analyzes retail sales data of electronic products across multiple cities using SQL, Excel, Python, and Tableau. The goal is to extract meaningful insights that help in improving business decisions related to sales, marketing, and product strategy.

---

## 🎯 Problem Statement
A retail company collects sales data but lacks insights into:
- Top-performing products
- Best-performing cities
- Category-wise revenue
- Sales trends over time

This project solves these problems using data analysis techniques.

---

## 🛠️ Tools & Technologies
- SQL Server (Data querying)
- Microsoft Excel (Data analysis & Pivot Tables)
- Python (Pandas for analysis)
- Tableau (Data visualization & dashboard)
- GitHub (Version control)

## 📊 Key Analysis Performed

### 🔹 Excel

- Pivot Table analysis  

### 🔹 Python
- Data loading & preprocessing  
- Summary statistics

### 🔹 Tableau

- Line chart → Sales trend
